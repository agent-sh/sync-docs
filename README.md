# sync-docs

Sync documentation with code changes. Find outdated refs, update CHANGELOG, flag stale examples.

## Installation

```bash
# Claude Code
claude mcp add-plugin agent-sh/sync-docs

# Or install from marketplace
agentsys install sync-docs
```

## Usage

```
/sync-docs
```

## Keywords

`documentation`, `changelog`, `sync`, `maintenance`, `code-docs`
